# Mind Forge App

## Product Brief + Specs + Phased Build Plan

## 1. Core idea

**Mind Forge** is a daily intellectual and reflective training app that helps a user grow **intelligence, wisdom, articulation, and personal depth** through structured conversations, prompts, and synthesis.

It is not a notes app.
It is not generic journaling.
It is not a quote-of-the-day toy.

It is a **cognitive training chamber** where the user:

* encounters a rich daily idea
* reflects on it
* connects it to real life
* articulates an original thought
* distills a portable insight they can use in conversation and action

The app should feel like:

* part philosopher’s workshop
* part mental gym
* part idea garden
* part personal intelligence operating system

# 7. Main experience loop

This is the heart of the app.

## Daily Mind Forge Session

1. User opens app
2. Receives a daily theme or question
3. Reads a rich prompt or idea seed
4. Reflects in writing
5. Extends the idea with their own connection / objection / analogy
6. Distills the thought into a concise “portable insight”
7. Optionally gets AI refinement / challenge / alternate lens
8. Saves insight to vault
9. Tracks streak, themes, and growth

This is the main loop:
**Encounter → Reflect → Extend → Distill → Apply → Archive**

# 8. Core modules

## 1. Daily Forge

The main daily session screen.

Contains:

* Daily idea/theme
* Rich seed passage or prompt
* Reflection field
* Extension field
* Distillation field
* Optional “challenge my thinking” AI button
* Save session

### Session structure

Each session should have these steps:

**Step 1: Spark**
A theme, paradox, question, quote, or scenario.

**Step 2: Reflect**
What does this mean? What feels true? What is interesting here?

**Step 3: Extend**
Add one of:

* analogy
* objection
* real-world connection
* personal application
* cross-domain link

**Step 4: Distill**
Complete:

* “The real thing here is…”
* or “Most people miss that…”
* or “This matters because…”

**Step 5: Apply**
Optional:

* Where does this show up in your life today?
* How might you use this in conversation?
* What decision does this affect?
